Network Security

This course provides a comprehensive introduction to network security principles, common attacks, and defensive mechanisms.

Exploits and defences in control hijacking attacks; the principle of least privilege, access control, and OS security; isolation and sandboxing; vulnerability testing using fuzzing, static, and dynamic analysis.

Basic web security model; web application security; web session management; Goals and pitfalls for HTTPS.
