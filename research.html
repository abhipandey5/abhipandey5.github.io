Current Research Projects

ML-Based Authentication for Healthcare Systems

This project focuses on developing advanced authentication techniques using machine learning to secure sensitive healthcare data. The goal is to create robust, adaptive, and user-friendly security systems that can protect against emerging threats in IoMT environments.

Security in Cyber-Physical Systems (CPS)

We are exploring the security and privacy challenges in Cyber-Physical Systems, particularly in critical sectors like healthcare and transportation, to ensure the integrity and confidentiality of data from interconnected physical devices.
